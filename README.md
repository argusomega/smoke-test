# smoke-test

A simple greeting CLI built with Python.

## Usage

```bash
python main.py
```

Output: `Hello, World!`

Customize the greeting with `--name`:

```bash
python main.py --name Alice
```

Output: `Hello, Alice!`

Customize the greeting itself with `--greeting`:

```bash
python main.py --greeting Hi --name Alice
```

Output: `Hi, Alice!`
